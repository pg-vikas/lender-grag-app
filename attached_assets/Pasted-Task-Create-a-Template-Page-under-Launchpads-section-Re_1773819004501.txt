Task: Create a **Template Page** under Launchpads section.

Requirements:

1. **Page Layout**
   * Page should follow **existing dashboard theme** (spacing, typography, colors, card styles).

2. **Template List**
   * Display templates in a **grid or list layout** as per screenshot.
   * Each template card should show:
     - Template name/title
     - Description or snippet
     - Creation/updated date
     - Action icons/buttons: **Edit**, **Delete**, **Apply**

3. **Add Template**
   * Include **“Add Template” button**.
   * Clicking opens a **popup/modal** to create a new template:
     - Fields: Template Name, Description, Category (optional)

4. **Edit Template**
   * Each template card has an **Edit icon**.

   * Each template card has a **Delete icon**.
   * Clicking opens a **confirmation popup**.
   * Confirm deletes the template; Cancel closes popup without changes.

6. **Apply Template**
   * Optional **Apply button** on each template card.
   * Clicking applies the template to the current launchpad or opens a selection popup if needed.

7. **UI & Behavior**
   * Popups, lists, and buttons must follow **dashboard theme/design**.
   * Page should handle **empty state** (no templates) gracefully.
   * All actions (Add/Edit/Delete/Apply) update the list **dynamically without page reload**.

Expected Result:

* Templates are displayed in a list/grid with action icons/buttons.
* Add/Edit/Delete popups work correctly and match **dashboard theme**.
