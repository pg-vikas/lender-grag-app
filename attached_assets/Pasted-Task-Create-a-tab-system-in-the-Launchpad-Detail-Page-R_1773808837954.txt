Task: Create a **tab system** in the Launchpad Detail Page.

Requirements:

1. **Tabs Layout**
   * Implement 5 tabs: **Chat, Public Notes, Private Notes, Uploads, Activities**.
   * Only one tab content should be visible at a time.
   * Active tab should be highlighted (different color, underline, or shadow) based on screenshot style.

2. **Tab Content (UI based on screenshot)**

   * **Chat**: 
     - Scrollable message list.
     - Input box to add a new message.
     - Show sender name, avatar, and timestamp.

   * **Public Notes**: 
     - Display notes in **card style** as in screenshot.
     - Each note shows title, content snippet, and creation date.
     - Add button to create a new note, opens modal or inline editor.

   * **Private Notes**: 
     - Same card style as Public Notes.
     - Only accessible to authorized users.
     - Add/Edit/Delete options as per screenshot layout.

   * **Uploads**: 
     - Show uploaded files in **grid/list layout** matching screenshot (preview thumbnails for images, icons for documents).
     - Include **upload button** for new files.
     - Allow delete or download actions as per screenshot.

   * **Activities**: 
     - Timeline or list of recent actions/events.
     - Include icons and timestamps for clarity.

3. **Navigation**
   * Clicking a tab switches content without page reload.
   * Tabs should be responsive and work on mobile & desktop.

4. **UI Consistency**
   * Match the screenshot style: spacing, card shadows, typography, colors.
   * Buttons, inputs, and cards should follow dashboard style.
   * Include hover and active effects consistent with other modules.

Expected Result:

* Clicking on a tab shows the **correct content** matching the screenshot.
* Public Notes, Private Notes, and Uploads match the **visual style of the screenshot**.
* Users can interact with notes and uploads seamlessly without page reload.
* Tabs are fully functional, responsive, and visually consistent with the dashboard.
