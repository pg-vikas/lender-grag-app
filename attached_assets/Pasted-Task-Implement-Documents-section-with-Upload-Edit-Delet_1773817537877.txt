Task: Implement **Documents section with Upload, Edit, Delete, and Add Category** in Launchpad Detail Page.

Requirements:

1. **Add Category**
   * Add **“Add Category” button** in the Documents section.
   * Clicking it opens a **popup/modal** to enter category name.
   * Popup has **Save/Submit** and **Cancel** buttons.
   * On **Save**, the new category is added to the list dynamically.

2. **Category List**
   * Each category in the list displays **three action icons**:
     - **Upload** → opens upload popup
     - **Edit** → opens edit category popup
     - **Delete** → opens delete confirmation popup

3. **Upload Popup Behavior**
   * Clicking **Upload** opens a **popup/modal** matching dashboard theme.
   * Popup includes:
     - File input
     - **Type dropdown**: Client Signed, Contractor Signed, Fully Executed, Not Signed (default)
     - **Save** and **Cancel** buttons
   * **Important:** Popup **does not close automatically** when file is selected; closes only when **Save** or **Cancel** is clicked.
   * On **Save/Submit**:
     - The uploaded file is added to the **correct category list**.
     - It appears immediately in the list (dynamic state update).
     - Other categories/lists remain unaffected.

4. **Edit Icon**
   * Opens **edit category popup**, allows updating category name/details.
   * Save updates category immediately; Cancel closes popup without changes.

5. **Delete Icon**
   * Opens **delete confirmation popup**, removes only selected category on confirm.

6. **UI & Behavior**
   * All popups and lists follow **dashboard theme/design**.
   * Lists update **dynamically** after upload, edit, or delete without page reload.
   * Only clicking the respective icons or buttons triggers popups; no other clicks should affect them.

Expected Result:

* Upload popup stays open until **Save** is clicked.  
* Document shows immediately in the correct category list after upload.  
* Edit/Delete popups function correctly.  
* All UI elements match existing dashboard styling and theme.
