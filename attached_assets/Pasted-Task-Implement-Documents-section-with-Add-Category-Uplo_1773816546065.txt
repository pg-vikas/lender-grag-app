Task: Implement **Documents section with Add Category, Upload, Edit, and Delete** in Launchpad Detail Page.

Requirements:

1. **Add Category**
   * Add **“Add Category” button** in the Documents section.
   * Clicking it opens a **popup/modal**.
   * Popup allows user to enter **category name**.
   * Popup has **Submit/Save** and **Cancel** buttons.
   * On Submit: the category is added to the list dynamically.

2. **Category List**
   * Each category displays in the list/grid.
   * Each category has **three action icons**:
     - **Upload** → opens upload popup
     - **Edit** → opens edit category popup
     - **Delete** → opens delete confirmation popup

3. **Upload Icon Functionality**
   * Clicking **Upload** opens a **popup** matching the dashboard theme.
   * Popup contains:
     - File input
     - **Type dropdown** with options:
       + Client Signed  
       + Contractor Signed  
       + Fully Executed  
       + Not Signed (default)
     - Submit and Cancel buttons
   * On Submit:
     - File is added to the **selected type** list under that category only.
     - Other types remain unaffected.
   * Uploaded files appear in a **list/grid** with file name, type, and action icons (edit/delete).

4. **Edit Icon Functionality**
   * Clicking **Edit** opens **edit category popup**.
   * Allows updating category name/details.
   * Popup has Save/Cancel buttons; changes reflect in category list immediately.

5. **Delete Icon Functionality**
   * Clicking **Delete** opens **delete confirmation popup**.
   * Confirm removes the category; Cancel closes popup without deleting.
   * Only affects the selected category.

6. **UI & Behavior**
   * All popups follow **existing dashboard theme** (colors, spacing, typography).
   * Popups overlay properly and are responsive.
   * Only clicking icons/buttons triggers respective popups; no other clicks should affect them.

Expected Result:

* **Add Category button** opens popup → submit adds category to list.
* **Each category** has Upload/Edit/Delete icons.
* Upload popup allows file selection + Type → adds file only to selected type.
* Edit popup updates category.
* Delete popup deletes category.
* UI and interactions **match the reference Launchpad document section** exactly.
