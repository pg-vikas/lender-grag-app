Build a production-style React document signing workflow for Gorilla Apps that feels as seamless and intuitive as DocuSign, but uses our own premium dark design system.

This is a document sending and preparation experience where users can:
- add recipients/signers
- upload one or more documents
- prepare the document by dragging fields onto pages
- assign fields to recipients
- configure email details
- review and send

The interface must feel polished, modern, premium, and very easy to use. It should not feel like a generic admin form. It should feel like a high-end e-sign product inside Gorilla Apps.

DESIGN SYSTEM / VISUAL STYLE:
- dark luxury UI
- deep navy background
- subtle gradients
- glowing purple accent
- glassmorphism cards
- rounded-xl or 2xl surfaces
- clean spacing
- premium typography
- elegant shadows
- smooth hover states
- subtle motion / micro animations
- highly usable, not cluttered
- responsive layout

TECH STACK:
- React
- TypeScript
- Tailwind CSS
- component-based architecture
- use local mock state only
- no backend required
- no real API calls
- no authentication logic
- no actual sending logic required
- if PDF rendering is too heavy, simulate document pages with styled page containers and realistic placeholder content
- build the architecture cleanly so real PDF rendering can be swapped in later

PRIMARY USER FLOW:
Convert the current signing wizard into this 5-step flow:

1. Add Recipients

TYPE DEFINITIONS:
Create and use these TypeScript types:

type StepId =
  | 'recipients'
  | 'documents'
  | 'prepare'
  | 'email'
  | 'review'

type RecipientRole =
  | 'Signer'
  | 'CC'
  | 'Approver'

type FieldType =
  | 'signature'
  | 'initials'
  | 'dateSigned'
  | 'fullName'
  | 'email'
  | 'title'
  | 'text'
  | 'checkbox'
  | 'arrow'

type Recipient = {
  id: string
  name: string
  email: string
  role: RecipientRole
  order: number
  color: string
}

type DocumentFile = {
  id: string
  name: string
  sizeLabel: string
  pageCount: number
  url?: string
}

type PlacedField = {
  id: string
  type: FieldType
  page: number
  x: number
  y: number
  width: number
  height: number
  recipientId?: string
  required: boolean
  label: string
  placeholder?: string
  fontSize?: number
  checked?: boolean
}

type EmailDetails = {
  subject: string
  message: string
  sendReminders: boolean
  reminderFrequency: 'daily' | 'every2days' | 'weekly'
  expiresInDays: number
}

type WizardState = {
  currentStep: StepId
  recipients: Recipient[]
  documents: DocumentFile[]
  placedFields: PlacedField[]
  selectedFieldId: string | null
  activeRecipientId: string | null
  emailDetails: EmailDetails
  zoom: number
}
