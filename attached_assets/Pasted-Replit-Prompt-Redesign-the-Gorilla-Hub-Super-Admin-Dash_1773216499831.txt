Replit Prompt

Redesign the Gorilla Hub Super Admin Dashboard with a bold, colorful, high-energy agency aesthetic while keeping the page highly functional for internal management.

The goal is to make the Super Admin area feel like a powerful control center rather than a generic admin template.

Design Direction

The visual tone should feel like:

a modern tech agency command center

bold but organized

colorful but controlled

visually exciting while still highly usable

Think of a mix between:

a high end SaaS dashboard

a creative agency design system

a futuristic control room

Avoid boring gray admin templates.

Core Color System

Use a dark foundation with bold color accents.

Background
Deep charcoal or midnight navy.

Accent colors for sections

electric purple

neon blue

hot magenta

cyan

vibrant orange

lime green

Each module should have a distinct color identity so the admin can quickly identify areas of the system.

Use gradients instead of flat colors.

Example gradient styles

purple → magenta
blue → cyan
orange → gold
green → teal

Glow Effects

Use subtle glow accents.

Example

Typography

Use a modern clean font.

Strong headings
clear hierarchy

Admin dashboards should feel precise and engineered.

Interaction Design

Add subtle motion.

Card hover lift

Everything should feel alive but not distracting.

Signature Gorilla Touch

Add subtle branding elements.

Examples

Gorilla icon accents

Final Goal

The Super Admin page should feel like:

“The command center of a powerful software ecosystem.”

Not a boring backend.

When someone opens Gorilla Hub Admin they should immediately think:
