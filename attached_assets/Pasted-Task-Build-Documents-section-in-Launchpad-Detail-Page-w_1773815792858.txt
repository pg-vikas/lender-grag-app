     - “Are you sure?” message
     - **Confirm/Delete** button → removes the document
     - **Cancel/Close** button → closes without deleting

5. **UI & Behavior**
   * All popups follow the **same design theme** as your Launchpad UI (colors, spacing, typography).
   * Popups overlay and close properly.
   * Document lists update **dynamically** after upload, edit, or delete without page refresh.

Expected Result:

* Upload popup opens with **Type dropdown**, saves document only in the selected category.

* UI matches existing Launchpad document section styling exactly.
